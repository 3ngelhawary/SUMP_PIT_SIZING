Warnings / Assumptions

Ready for calculation.
    • Pump discharge rate per pump is automatically set to inflow ÷ duty pumps.
    • Total duty pumping rate therefore equals incoming flow.
    • Minimum active volume is calculated using Vmin = Tmin × Q / 4.
    • Tmin = 3600 / starts per hour.
    • Plan area = active volume / effective depth.
    • Total wet well volume includes dead storage + effective depth + freeboard.
