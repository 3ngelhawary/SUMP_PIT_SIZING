Wastewater Pump Station / Sump Pit Sizing App
============================================

Deployment
----------
1. Upload the full folder to a GitHub repository.
2. Keep index.html in the repository root.
3. Commit and push.
4. In GitHub repository settings, open Pages.
5. Publish from the main branch root.

Notes
-----
- SI units only.
- Frontend only.
- No backend.
- Uses HTML, CSS, and vanilla JavaScript modules.
- Open index.html directly in a browser to run locally.
